# Contributing to AISpark Studio

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run backend: `cd backend && python -m uvicorn main:app --port 8001 --reload`
5. Run frontend: `cd nextjs-frontend && npm run dev`
6. Verify: Check `/docs` endpoint for API documentation
7. Commit with a descriptive message
8. Open a Pull Request

## Code Style

- Python: Follow PEP 8, use type hints
- TypeScript: Strict mode, follow existing patterns
- Write descriptive commit messages

## Reporting Issues

Use GitHub Issues. Include steps to reproduce, expected behavior, and actual behavior.
